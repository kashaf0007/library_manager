# library_manager 🚀
Personal Library Manager
# Core Features 
## 1- 📚 Book Details:
Title (string)

Author (string)

Publication Year (integer) 

Genre (string) 

📂 Read Status (boolean: True if read, False if unread).

## 2- Menu System: 🔍
Add a book 

Remove a book 

Search for a book 

Display all books 

Display statistics (total books, percentage read) ,

Exit .
